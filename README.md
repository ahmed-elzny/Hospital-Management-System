# 🏥 Hospital Management System

A Hospital Database Management System developed using Python, Tkinter, and SQLite.

## 📌 Overview

This project is a desktop application for managing a hospital database. It allows users to perform complete CRUD operations on different hospital entities through a graphical user interface.

## ✨ Features

- Manage Doctors
- Manage Patients
- Manage Appointments
- Manage Prescriptions
- Manage Medications
- Insert Records
- Update Records
- Delete Records
- Search Records
- SQLite Database Integration
- User-Friendly GUI

## 🛠 Technologies Used

- Python 3
- Tkinter
- SQLite3

## 📂 Project Structure

```
Hospital-Management-System/
│
├── hospital_gui.py
├── hospital.db
├── hospital_db.sql
├── README.md
└── .gitignore
```

## 🚀 How to Run

1. Clone the repository

```
git clone https://github.com/ahmed-elzny/Hospital-Management-System.git
```

2. Open the project folder

3. Run

```
python hospital_gui.py
```

No external libraries are required because Tkinter and SQLite3 are built into Python.
